Fixtures

Controls how far ahead the dashboard loads fixtures (saved in this browser). Use 7 days during UEFA cup finals so late-week matches appear. Changing this reloads the dashboard with ?days=5 or ?days=7; the server also remembers your choice in a cookie. Server default without a choice: {{ fetch_days }} days (env HIBS_FETCH_DAYS optional).
